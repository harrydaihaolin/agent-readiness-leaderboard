Scores updated daily at 06:00 UTC via GitHub Actions · View on GitHub · Scoring methodology

Each repo is scored out of 100 points across 7 categories:

  • AI Context Files (25 pts) — CLAUDE.md, AGENTS.md, .cursorrules, copilot instructions
  • Documentation (20 pts) — README, CONTRIBUTING, CHANGELOG, SECURITY, CODE_OF_CONDUCT
  • Testing (15 pts) — test directories and test runner configs (pytest, jest, vitest…)
  • CI/CD (15 pts) — GitHub Actions workflow count and depth
  • Tooling (10 pts) — linting, formatting, and type-checking configs
  • Code Structure (10 pts) — language count, topics, and license
  • Conventions (5 pts) — issue/PR templates and community standards

File detection uses the GitHub Git Trees API (1 call per repo) to avoid rate limits. Seed data is approximate; live scores are computed by the scanner.
