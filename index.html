Powered by agent-readiness · leaderboard source · Scores updated daily at 06:00 UTC via GitHub Actions · Scoring methodology

Each repo is cloned and scored 0–100 by the agent-readiness CLI tool across 4 pillars:

  • 🧠 Cognitive Load — README clarity, run instructions, onboarding friction
  • 🔄 Feedback Loops — testing coverage, CI checks, evaluation harnesses
  • ⚡ Flow & Reliability — automation, dependency health, contributor experience
  • 🛡️ Safety — security advisories, dependency scanning, responsible-AI signals

Repos are shallow-cloned (git clone --depth 1) in parallel during each CI run. Rank always reflects the overall agent-readiness score.
